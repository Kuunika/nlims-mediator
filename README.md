# Kuunika - Data for Action NLIMS Mediator

## Installation

```bash
$ npm install
```

## Config
1. Copy .env.example to .env
2. Set the values for the keys in .env

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
